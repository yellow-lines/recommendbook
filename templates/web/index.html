Зачем?

Мы читаем по разным причинам: для получения самой разной информации, для отдыха, для расширения кругозора. Есть множество причин, почему мы должны читать книги. Хорошая литература — это жизнь. Часто мы ищем в ней то, чего нам крайне не хватает в реальности. Происходящее в нашей душе, попытка ответить на интересующие нас вопросы – все это открывает нам книга. Задача рекомендательной системы – проинформировать пользователя о книге, которая ему может быть наиболее интересен в данный момент времени. Но как же лучше всего организовать подбор хорошей и полезной литературы для каждого пользователя?

Наш инновационный подход

База знаний — база данных, содержащая правила вывода и информацию о человеческом опыте и знаниях в некоторой предметной области. В самообучающихся системах база знаний также содержит информацию, являющуюся результатом решения предыдущих задач. Так же база знаний — важный компонент интеллектуальной системы. Наиболее известный класс — это экспертные системы. Главная цель создания баз знаний — помочь менее опытным людям найти уже существующее описание способа решения какой-либо проблемы. В нашем случае помочь новым пользователям подобрать для них книги, которые с большой вероятностью им подойдут. Веб-сайт с базой знаний является более инновационным. Он предоставляет пользователям место для самостоятельного поиска ответов на вопросы, связанный с выбором подходящей литературы. Системы баз знаний — отличное решение этой проблемы. Пользователям предоставляется удобный сервис самопомощи.
